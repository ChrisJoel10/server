LangCorrect
^^^^^^^^^^^
`LangCorrect <https://langcorrect.com/>`_ is a platform that allows users to practice and improve their writing skills in a foreign language.
Through the platform, users can submit text passages and receive corrections from native speakers.
This documentation aims to provide comprehensive guides and API references to help you understand and work efficiently with LangCorrect.
